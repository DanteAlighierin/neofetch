# neofetch
Yes, this is fork. This is the fork of MacOs' version of neofetch. I've ported to my linux desktop.

## Installation
```bash
git clone https://github.com/DanteAlighierin/neofetch.git
```

```bash 
cd neofetch
```

```bash
python3 neofetch.py
```
